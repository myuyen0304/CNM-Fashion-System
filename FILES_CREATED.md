```

→ LoginPage.jsx
 → axiosClient.js (POST /api/auth/login)
 → auth.controller.js
 → auth.service.js
 → auth.model.js (User schema)
```

### Shopping Flow

```
HomePage.jsx / SearchPage.jsx
→ ProductCard.jsx (display)
→ ProductDetailPage.jsx
 → CartContext.jsx (addItem)
 → axiosClient.js (POST /api/cart/items)
 → cart.controller.js
 → cart.service.js
 → cart.model.js (Cart schema)
```

### Checkout Flow

```
CartPage.jsx
→ CheckoutPage.jsx (3-step form)
 → axiosClient.js (POST /api/orders)
 → order.controller.js
 → order.service.js
 → payment.service.js
 → VNPAY API
 → PaymentResultPage.jsx
```

```

→ (Review Form)

 → review.controller.js

 → review.model.js
```
